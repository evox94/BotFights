About:

Hi my name is "Botfights" and I am a programing challenge platform that programmers can use to practise their programming skills and engage each other in AI battles.
You choose a game, load up your source code and witness your bots duking it out in a virtual arena.

I also represents a framework for developing said games.

I was developed in Java and I use JavaFX as my graphics framework.

Checkout my github:

How to:

Firstly, you can always go back to this page by using menu Help -> About.

On the left you can see a list of loaded games, and you can choose one by clicking on it.
When you do that a game description will load and you can check that out to learn more about how the game is played.

Below the game list you will see a 'Templates' section with its title written in pink font.
You can use this section to download a "starter" file in one of the programming languages the selected game supports.
This file will help you in writing your own bot.

Bellow 'Templates' section, occupying bottom of the screen is the, 'Simulator console'.
Here you can see all the important info and errors that can popup, so if you are not sure why something happened
during the run time of the simulator, you can probably find the explanation in here.

On the right you have the 'Player manager' section. Here you can add/remove players, if the game supports variable number of players.
Each player is represented by his name and his source fil containing the bot source code.

After you input those, for each player, a green 'Launch' button will light up and after that i guess you know what to do.
